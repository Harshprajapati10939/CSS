Element selector

Element selector basically element/tag directly,selected elements based on name,id,class
example-Select by Name: 

p{
                text-align:center;
                color:red;
            }
exmple-select by id: 
#para1{
                text-align:center;
                color:red
            }
exmple-select by class: 
.para1{
                text-align:center;
                color:red
            }
